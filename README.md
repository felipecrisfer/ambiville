O Ambiville é um projeto desenvolvido com o objetivo de incentivar o descarte correto de resíduos na cidade de Joinville.
A plataforma permite que os cidadãos encontrem locais adequados para descartar diferentes tipos de resíduos e também solicitem a retirada de materiais diretamente em suas residências.

O projeto busca facilitar o acesso a informações sobre reciclagem e contribuir para uma cidade mais limpa e sustentável.

Foi desenvolvido uma página que mostra os indicadores das mudanças causadas pelo sistema na cidade, também criado um sobre para que tivesse curiosidade sobre o projeto pudesse saber um pouco mais!
